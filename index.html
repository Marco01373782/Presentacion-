TITULO DEL PROYECTO

PAGINA WEB, USANDO HTML Y CSS, PARA ORGANIZACIÓN Y CONTROL DE MATERIALES Y HERRAMIENTAS DE TRABAJO.

RESUMEN

La organización es definida como el proceso para llevar a cabo deferentes actividades, así como también el uso y control eficiente de los materiales y herramientas empleadas. Uno de los problemas que nos encontramos cuando un taller se amplia es la falta de organización y orden de su materia prima y herramientas, por tal razón la falta de un sistema automatizado que realice dichas tareas es crucial de modo que la entidad que lo está empleando (taller o empresa) no pierda tiempo y dinero. El objetivo general es desarrollar una página web, usando programación en HTML y CSS, para organización y control de materiales y herramientas de trabajo. La presente página web será desarrollada mediante y siguiendo las fases de la metodología scrum, que es un proceso que se lo ha utilizado regularmente para trabajar en equipo y obtener el mejor resultado al final del proyecto, este a su vez tiene algunos pasos a seguir para realizarlo de manera exitosa y correcta. Planificación: Product Backlog, se establecen las tareas prioritarias. Ejecución: Sprint, es como un mini proyecto en donde el equipo de trabajo se enfoca en el desarrollo de tareas para lograr el objetivo que se ha definido previamente. Control: Burn Down, se mide el avance de un proyecto conforme al tiempo empleado y los objetivos propuestos. Para finalizar, los resultaos esperados al finalizar son ampliar los conocimientos adquiridos en las diferentes asignaturas y que la página web sea de ayuda para el usuario enfocado.

DIAGNOSTICO

El desarrollo de un sitio web para organización y control de herramientas está enfocado en ayudar a negocios con una problemática en general, la falta de control y organización de material de trabajo, por eso este proyecto busca solucionarlo, para lo cual utilizamos el sistema de programación HTML y CSS que nos permite desarrollar en el sitio web como un controlador de inventario que puede ser usado por operadores o propietarios lo que les ayudara a saber cuántas herramientas de trabajo disponen y cuanto materia tiene a su disponibilidad. El programa Control de Herramientas permite administrar resguardos y devoluciones de todas las herramientas entregadas a los trabajadores. El programa permite consultar en línea quién tiene o dónde se encuentra cada una de las herramientas entre otras cosas. Antes de proceder a efectuar un trabajo de mantenimiento, desde el código el usuario podrá consultar la disponibilidad o existencias en el almacén de las herramientas que empleará para realizar los trabajos encomendados, con este práctico sitio web podrás mantener organizada y al día toda la información referente a las herramientas y materiales que cada empleado tiene en resguardo. Experiencias realizadas anteriormente para resolver el problema identificado. Muchas compañías de la construcción en el entorno no tienen consolidado sistemas de inventarios eficientes que se manejen homogéneamente dentro de todos sus proyectos, la gestión y sistemas de inventarios, stock, sistema ABC y manejo de almacenes, la gestión de inventarios para mejorar el control interno, supervisar el sistema, controlar los procedimientos y registros implementados y validar los procesos propuestos mediante indicadores de gestión. ENCUESTA APLICADA.

JUSTIFICACION

¿Qué intereses o necesidades sociales, tecnológicas o académicas pueden ser cubiertas con la investigación? Dentro del área de construcción, los talleres pequeños no se encuentran ajenos al problema de la falta de control y organización de materiales como herramientas de trabajo, lo cual, aunque no cause un gran impacto, a la larga si afecta el desempeño y desenvolvimiento de las tareas diarias realizadas dentro del cuerpo de trabajo, generando una pérdida de tiempo lo que conlleva a una pérdida de dinero para institución de trabajo. En la actualidad toda empresa y taller debería contar con un sistema de control automatizado que permita que tanto al jefe como los empleados mantener el área de trabajo siempre ordenada, de tal modo que se a posible completar los proyectos de forma rápida y de la mejor forma posible. La presente investigación es viable, pues se dispone de las herramientas de trabajo como la internet, encuestas y entrevistas, así también con recursos como fuentes de información, docentes capacitados y ordenadores a nuestra disposición. En el aspecto social, una pagina web para organización y control de materiales y herramientas de trabajo, busca facilitar la labor que debe realizar el personal del taller, con el fin de agilizar las tareas, de esta manera el taller será mucho más productivo, lo que conlleva a obtener una mayor ganancia y así, proyectarse a mercados más grandes contando ya con cimientos sólidos. La investigación cuenta con una utilidad metodología, ya que podrá utilizarse como base para futuras investigaciones que usen metodologías conciliables, de este modo serán posibles análisis conjuntos, comparaciones de resultados o entre periodos de tiempo, así como también evaluaciones de las operaciones que se estuvieran realizando. En el aspecto académico, es estudio pretende poner en practica los conocimientos adquiridos en cada una de las asignaturas tomadas, así como también presentar el desenvolvimiento que mostremos de acuerdo a una problemática del mundo real que requiere de una solución desde el ámbito informático y tecnológico.
